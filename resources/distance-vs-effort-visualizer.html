Distance vs Effort Visualizer

Interactively visualize how varying the visual distance target (quality) and encoder effort affect image appearance with the official JPEG XL reference encoder.


Size: bytes

Encoding Time: s

Distance quantifies visual distortion using the psychovisually robust Butteraugli metric, where zero represents mathematically lossless compression.

Effort controls the amount of computing power (CPU and memory) used to optimize compression efficiency while maintaining the target distance.

Encoding Time: Images were compressed using libjxl 0.10.2 on a 10-Core, 32GB, M1 Max MacBook Pro on June 14, 2024.
